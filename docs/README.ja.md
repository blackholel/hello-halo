## なぜKite？

友人に最新のAIツールを勧めるたびに、同じ問題に直面します：

> "良さそうだけど、コマンドラインの使い方がわからない..."
> "Node.jsをインストール？それって何？"
> "やめておくよ、複雑すぎる。"

**AI技術は急速に進歩していますが、ほとんどの人が取り残されています。**

**Kiteはこれらの障壁を打ち破るために作られました。**

---

## Kiteとは？

Kiteは**すぐに使えるAIアシスタントデスクトップアプリケーション**です。

世界最強のAIコーディングアシスタント（Claude Code）を、誰でも使えるグラフィカルインターフェースでラップしています。

**超賢いアシスタントと話すのと同じくらい簡単です。**

「ダークモード付きのミニマルな個人ウェブサイトをデザインして」

「画像をアップロードするたびに、自動的に圧縮してWebPに変換して」

## クイックスタート

### 1. インストール

| プラットフォーム | ダウンロード | 要件 |
|-----------------|-------------|------|
| **macOS** (Apple Silicon) | [.dmgをダウンロード](https://github.com/blackholel/buddykite/releases/latest) | macOS 11+ |
| **Windows** | [.exeをダウンロード](https://github.com/blackholel/buddykite/releases/latest) | Windows 10+ |
| **Linux** | [.AppImageをダウンロード](https://github.com/blackholel/buddykite/releases/latest) | Ubuntu 20.04+ |

### 2. APIキーを取得

1. [Anthropic Console](https://console.anthropic.com/)にアクセス
2. APIキーを作成
3. Kiteの設定にキーを貼り付け

### 3. 作成を始める

```
"こんにちは！シンプルな電卓のウェブページを作って"
```
